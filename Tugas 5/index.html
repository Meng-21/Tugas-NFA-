About Me

Perkenalkan, nama saya Nurmayanti, mahasiswa Program Studi Teknik Informatika di STT Nurul Fikri. Saat ini saya telah memasuki semester 6 dalam perjalanan akademik saya. Sejak awal kuliah, saya memiliki ketertarikan yang besar terhadap dunia teknologi informasi, khususnya dalam pengembangan sistem berbasis web dan penerapan metode kerja yang terstruktur.

Pada semester 5, saya berkesempatan mengikuti program magang di sebuah software house dengan posisi sebagai Scrum Master. Dalam peran tersebut, saya bertanggung jawab untuk memfasilitasi tim pengembang agar dapat bekerja secara efektif menggunakan kerangka kerja Agile Scrum. Jobdesk saya meliputi memimpin daily stand-up, menyusun dan memantau sprint backlog, memastikan komunikasi antar anggota tim berjalan lancar, serta membantu mengatasi hambatan yang muncul selama proses pengembangan. Pengalaman ini memberikan saya wawasan berharga tentang bagaimana sebuah tim teknologi berkolaborasi untuk menghasilkan produk yang berkualitas.
